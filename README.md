<h1><b>Exercise 2:</b></h1>
1. Form Groups of 2 - 4<br>
2. One person of the group: Fork our Repository: (link) /need to create that; Simple HTML website<br>
3. Invite all members to the repository<br>
4. Everyone clones the repository of the forked repo to their laptop<br>
5. Everyone creates a branch for their own, with their name<br>
6. In your branch adjust the file “best_movie.html” to your liking<br>
7. Merge your branch into the main branch<br>
8. Work out the merge conflicts as a group<br>
9. Accept the merge request<br>
10. Delete all branches except the main branch<br>
